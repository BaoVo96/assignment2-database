Github repository này chứa toàn bộ souce code của nhóm em của bài tập lớn 2. Gồm các file sau đây:

1. Full.sql: Chứa toàn bộ code.
2. create_tables.sql: Chứa các câu lệnh để tạo ra các bảng
3. insert_data.sql: Chứa toàn bộ các câu lệnh để chèn dữ liệu vào các bảng
4. add_foreign_key.sql: Chứa toàn bộ các câu lệnh để thêm foreign key vào các bảng
5. create_index.sql: Chứa toàn bộ các câu lệnh để tạo index
6. user.sql: Chứa các câu lệnh để tạo user, gán quyền cho user cũng như một số câu lệnh để minh họa cho việc cấp quyền user
7. queries.sql: Chứa toàn bộ các câu lệnh query mà nhóm đã viết ra
8. add_trigger.sql: Chứa toàn bộ các câu lệnh để tạo ra các trigger
